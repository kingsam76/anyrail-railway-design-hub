## Overview

AnyRail is a Windows track-planning application for model railways, built around a visual workflow that keeps layout creation straightforward. It lets you place track, verify fit and spacing, and refine ideas before turning them into a physical build.

The software is a good fit for hobbyists and planners who need to compare parts, keep track of revisions, and generate shareable project output from one place. With 3D preview, export support, and multi-monitor operation, it scales from quick concept sketches to more detailed planning sessions.

---

## What it offers

- Build model railway layouts with a layout-first planning interface
- Check clearances as you position track sections
- Inspect designs with the integrated 3D preview
- Narrow parts by brand, scale, and part number
- Step backward and forward through changes with undo and redo
- Export plans as DXF, PDF, or SVG
- Work in multiple languages
- Use multi-monitor setups for a broader planning workspace

## Using AnyRail

A common approach is to begin a fresh layout, add track pieces, and arrange them on the canvas until the design fits your room or available space. While building the plan, use clearance checking and the 3D preview to review geometry and spacing before exporting the finished layout.

Typical export targets include:
- DXF for CAD-oriented workflows
- PDF for review and printing
- SVG for scalable vector output

You can also filter the component catalog by brand, scale, or part number to make selection faster during layout work.
